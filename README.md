# 숫자 야구 게임
## 구현목록

* 게임참여자 입력로직 추가
* 결과 세팅, 결과값 노출 로직 추가
* 값 체크로직 추가
* 입력값 validate 로직 추가
* 오류 메세지 세팅
* 테스트 케이스 작성

## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.
